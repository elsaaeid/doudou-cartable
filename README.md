# Doudou & Cartable

**Doudou & Cartable** is a WordPress project aimed at enhancing the educational experience for schools, teachers, and students. This website provides a platform for managing educational content, facilitating communication, and improving resource accessibility.

## Live Website

You can access the live website here: [Doudou & Cartable](https://doudou-cartable.netlify.app)

## Features

- **User-Friendly Interface**: An intuitive design that simplifies navigation and usability for all users.
- **Content Management**: Organize and manage educational materials, including documents, images, and videos.
- **Customizable Settings**: Tailor the website to meet the specific needs of your educational institution.
- **Responsive Design**: Fully responsive layout that ensures accessibility on all devices.
- **Multilingual Support**: Supports multiple languages, making it suitable for diverse educational environments.

## Installation

If you want to run the Doudou & Cartable project locally, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/yourusername/doudou-cartable.git
   cd doudou-cartable
   ```
   ## Set Up the Environment

1. Ensure you have **XAMPP** or any local server environment set up.
2. Create a new database in your local MySQL server.

2. **Configure WordPress**:

1. Download the latest version of **WordPress**.
2. Copy the WordPress files into your project directory.
3. Rename `wp-config-sample.php` to `wp-config.php` and update the database credentials.

3. **Install the Plugin**:

1. Navigate to the **Plugins** section in your WordPress dashboard.
2. Upload and activate the **Doudou & Cartable** plugin from the `wp-content/plugins` directory.

4. **Activate the Theme**:

1. If a custom theme is available, activate it in the **Appearance > Themes** section.

5. **Configure Settings**

1. Navigate to the **Doudou & Cartable** settings page to customize the website according to your needs.

## Usage

After setting up the website, you can access various features, such as:

- Adding and managing educational resources.
- Organizing materials into categories for easy access.
- Customizing user roles and permissions for different user types.
- Generating reports on resource usage and engagement.
